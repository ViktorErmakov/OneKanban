Работа с доской

Панель управления​

На верхней панели доски расположены все инструменты для работы с канбан доской: быстрые отборы, фильтры, группировка, поиск, переключение темы и кнопки управления.

Быстрые отборы по проектам​

На панели управления расположен выбор проектов в виде цветных «пилюль» с названиями выбранных проектов. Если выбрано больше трёх проектов, оставшиеся скрываются под счётчиком +N.

Нажатие на пилюлю убирает проект из отбора, а нажатие на стрелку открывает выпадающий список всех доступных проектов.

В списке можно выбрать один или несколько проектов, а также использовать пункт Выбрать все для быстрого выделения всех проектов.

Быстрый отбор по проекту с карточки

Нажмите на цветную метку проекта на любой карточке — доска переключится на отображение только задач этого проекта. Повторное нажатие снимает отбор.

Фильтр по исполнителям​

Фильтр Исполнитель позволяет отобрать задачи по одному или нескольким исполнителям. В выпадающем списке отображаются фотографии и имена всех исполнителей, задачи которых есть на доске.

  • Выберите одного или нескольких исполнителей
  • Кнопка × рядом с названием фильтра сбрасывает отбор
Быстрый отбор по исполнителю с карточки

Нажмите на фотографию исполнителя на любой карточке — доска отфильтрует задачи по этому исполнителю. Повторное нажатие снимает отбор.

Фильтр по срочности​

Фильтр Срочность позволяет отобрать задачи по уровню срочности. Уровни срочности задаются в 1С и отображаются в выпадающем списке с настроенными иконками и цветами.

  • Выберите один или несколько уровней срочности
  • Кнопка × сбрасывает отбор
Быстрый отбор по срочности с карточки

Нажмите на иконку срочности на любой карточке — доска отфильтрует задачи по этому уровню срочности. Повторное нажатие снимает отбор.

Настройка иконок срочности​

У каждого уровня срочности в выпадающем списке есть кнопка ⚙ (настройка). Нажмите на неё, чтобы открыть попап с выбором иконки и цвета для данного уровня.

Доступные иконки: двойная стрелка вверх, стрелка вверх, равно, стрелка вниз, флаг, молния, восклицательный знак.

Доступные цвета: красный, оранжевый, янтарный, жёлтый, зелёный, синий.

После выбора нажмите Сохранить для применения или Отмена для отката изменений.

Фильтр по типу карточки​

Фильтр Тип позволяет отобрать задачи по типу: Задача или Ошибка. Поддерживается множественный выбор.

Группировка​

Доска поддерживает группировку задач по двум критериям:

  • По исполнителю — задачи группируются по исполнителям с отображением фотографии и имени
  • По проектам — задачи группируются по проектам с отображением цвета проекта

Каждая группа содержит счётчик задач и может быть свёрнута/развёрнута нажатием на заголовок группы.

При группировке по проектам перетаскивание задач между группами запрещено. При группировке по исполнителям перетаскивание между группами меняет исполнителя задачи.

Для снятия группировки выберите Без группировки.

Поиск по задачам​

Поле Поиск по задачам... фильтрует карточки по введённому тексту в реальном времени. Поиск выполняется по текстовому описанию задачи. Для очистки поиска нажмите кнопку ×.

Переключатель темы​

Кнопка переключения темы расположена в правой части панели управления. Поддерживаются светлая и тёмная темы. Выбранная тема сохраняется между сессиями.

Добавление задач​

Если в настройках доски для статуса поставили галочку Кнопка добавить задачу, то в колонке этого статуса будет отображаться кнопка, по нажатию на которую будет открыто окно создания новой задачи.

Drag & Drop​

Перетаскивание карточек между колонками изменяет статус задачи. При перетаскивании целевая колонка подсвечивается.

В режиме группировки Drag & Drop также работает:

  • Группировка по исполнителям: можно перетаскивать карточки между группами — при этом меняется исполнитель задачи
  • Группировка по проектам: перетаскивание между группами разных проектов запрещено (колонка подсвечивается красным)

Автолимит карточек​

Если в колонке слишком много карточек и они не помещаются по высоте экрана, лишние карточки автоматически скрываются. Внизу колонки появляется ссылка Ещё N, по нажатию на которую показываются все карточки.

Обновление отображения задач на доске​

Кнопка Обновить обновляет отображение задач на доске, если по каким-то причинам этого не произошло автоматически.

Настройки​

Кнопка Настройки вызывает окно настроек.
