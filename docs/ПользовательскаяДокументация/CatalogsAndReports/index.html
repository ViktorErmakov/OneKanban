Справочники и отчёты

Раздел OneKanban в интерфейсе 1С содержит не только канбан-доску, но и справочники для ведения данных и отчёт для анализа времени в статусах.

Настройка колонок доски, проектов и сопоставления статусов описана в Настройках доски. Работа с фильтрами и карточками на доске — в Работе с доской.

Справочники​

Задачи​

Основной объект учёта работ. Задачи попадают на доску, если их статус и проект входят в настройки доски.

Для встроенных объектов OneKanban в конфигурациях на БСП действует иерархия «проект → эпик → подзадачи» — см. Схему работы.

Открыть задачу можно из списка Задачи в подсистеме или кликом по номеру на карточке доски.

Ошибки​

Справочник ошибок используется вместе с сопоставлением статусов: ошибки отображаются в тех же колонках, что и задачи, с соответствующим статусом. На доске ошибки отличаются подписью «Ошибка №…» и красной полосой слева на карточке.

Проекты и статусы​

  • Проекты — группировка задач; цвет проекта задаётся в настройках доски
  • Статусы задач и Статусы ошибок — колонки доски и значения для сопоставления

Отчёт «История задач»​

Отчёт История задач (канбан_ИсторияЗадач) показывает, сколько времени задачи и ошибки провели в каждом статусе. Данные накапливаются при записи элементов (регистр истории статусов).

Отчёт доступен в подсистеме OneKanban — команда История задач (раздел «История задач» в вариантах отчётов БСП).

Варианты отчёта​

ВариантСодержание
ДетальноПоэтапная история смены статусов по каждой задаче
Сводка по задачеСуммарное время в каждом статусе в разрезе задачи
Сводка по статусуСуммарное время, проведённое задачами в каждом статусе
Сводка по исполнителюВремя по исполнителям и статусам
Сводка по проектуВремя по проектам и статусам
к сведению

Состав полей и отборов зависит от конфигурации-носителя. В нестандартных внедрениях текст запроса отчёта может переопределяться в модуле канбан_КанбанДоскаПереопределяемый.

HTML-редактор задач​

Для справочника Задачи OneKanban (в сборке с собственными объектами) доступен HTML-редактор описания задачи. Демонстрация — в блоге о релизе 2.5.
