## Что нового

### Новая панель управления

Панель управления полностью переработана. Теперь все инструменты — отборы, фильтры, группировка, поиск и настройки — расположены в одной компактной строке и работают без перезагрузки страницы.

### Фильтр по исполнителям

Появился полноценный фильтр по исполнителям с поддержкой множественного выбора. В выпадающем списке отображаются фотографии и имена исполнителей. Можно быстро отфильтровать задачи, нажав на фотографию исполнителя прямо на карточке.

### Фильтр по срочности

Новый фильтр по уровню срочности задач. Каждый уровень отображается с настраиваемой иконкой и цветом. Прямо из выпадающего списка можно настроить внешний вид иконки и цвет для каждого уровня срочности — выбрать из 7 иконок и 6 цветов.

### Фильтр по типу карточки

Добавлен фильтр по типу: **Задача** или **Ошибка**. Удобно для досок, где одновременно отображаются и задачи, и ошибки (например, в СППР).

### Группировка

Задачи на доске теперь можно группировать:

- **По исполнителю** — каждая группа показывает фотографию и имя исполнителя с количеством задач. При перетаскивании карточки в другую группу исполнитель автоматически меняется.
- **По проектам** — каждая группа показывает цвет и название проекта. Перетаскивание между группами разных проектов запрещено.

Группы можно сворачивать и разворачивать нажатием на заголовок.

### Поиск по задачам

Новое поле поиска фильтрует карточки в реальном времени по тексту описания задачи. Работает мгновенно, без перезагрузки страницы.

### Тёмная тема

Добавлена поддержка тёмной темы с переключателем на панели управления. Выбранная тема сохраняется между сессиями.

### Обновлённый выбор проектов

Выбор проектов переработан: теперь выбранные проекты отображаются в виде цветных «пилюль» на панели. Если выбрано много проектов, лишние скрываются под счётчиком. Можно быстро отфильтровать по проекту, нажав на цветную метку карточки.

### Автолимит карточек

Если в колонке слишком много карточек и они не помещаются по высоте экрана, лишние автоматически скрываются. Внизу колонки появляется ссылка «Ещё N» для отображения всех.

### Улучшенный Drag & Drop

Drag & Drop полностью переработан и теперь поддерживает все режимы — обычный и с группировкой. При группировке по проектам запрещённые зоны перетаскивания подсвечиваются.

### Интерактивные карточки

Элементы карточки стали интерактивными:
- Нажатие на **фото исполнителя** — быстрый фильтр по исполнителю
- Нажатие на **цветную метку проекта** — быстрый фильтр по проекту
- Нажатие на **иконку срочности** — быстрый фильтр по срочности
