## Панель управления

На верхней панели доски расположены все инструменты для работы с канбан-доской: быстрые отборы, фильтры, группировка, поиск, переключение темы и кнопки управления.

Выбранные отборы, группировка и тема **сохраняются для текущего пользователя** между сессиями. Текст поиска действует только до закрытия формы доски.

![Панель управления](/img/ControlPanel.png)

## Карточка на доске

Каждая карточка содержит:

- **Цветную метку проекта** — слева в шапке; при наведении показывается название проекта
- **Ссылку с номером** — «Задача №…» или «Ошибка №…»; по клику открывается форма элемента в 1С (ссылка не перетаскивается)
- **Фотографию исполнителя** — справа в шапке, если включено в [персональных настройках](./userSettings.md#персональные-настройки)
- **Иконку срочности** и **текст описания** — в теле карточки

У **ошибок** слева отображается красная полоса (класс `card-type-bug`), подпись начинается с «Ошибка №…».

![Состав карточки](/img/CardAnatomy.png)

:::tip Быстрые отборы с карточки

- **Метка проекта** — отбор по проекту
- **Фото исполнителя** — отбор по исполнителю (если фильтр виден)
- **Иконка срочности** — отбор по уровню срочности

Повторное нажатие снимает отбор.

:::

Клик по номеру задачи или ошибки открывает карточку в 1С:

![Открытие задачи по ссылке](/img/CardLinkOpen.png)

## Быстрые отборы по проектам

![Быстрые отборы](/img/QuickSelections.png)

На панели управления расположен **выбор проектов** в виде цветных «пилюль» с названиями выбранных проектов. Число видимых пилюль настраивается в выпадающем списке проектов (**от 1 до 5**); остальные выбранные проекты скрываются под счётчиком **+N**.

Нажатие на пилюлю убирает проект из отбора, а нажатие на стрелку открывает выпадающий список всех доступных проектов.

![Выпадающий список проектов](/img/ListOfProjects.png)

В списке можно выбрать один или несколько проектов, а также использовать пункт **Выбрать все** для быстрого выделения всех проектов.

Если в настройках доски для проектов не выбраны цвета, «пилюли» будут серыми.

![Выпадающий список проектов](/img/ListOfProjectsNoColor.png)

## Фильтр по исполнителям

![Без исполнителя](/img/ExecutorNoAssignee.png)

Фильтр **Исполнитель** позволяет отобрать задачи по одному или нескольким исполнителям. В выпадающем списке отображаются фотографии и имена исполнителей, задачи которых есть на доске. Первым пунктом идёт **Без исполнителя** — для карточек без назначенного исполнителя.

- Выберите одного или нескольких исполнителей
- Кнопка **×** рядом с названием фильтра сбрасывает отбор

Если на доске у всех карточек один и тот же исполнитель, фильтр **скрывается** автоматически.

## Фильтр по срочности

![Фильтр по срочности](/img/FilterUrgency.png)

Фильтр **Срочность** позволяет отобрать задачи по уровню срочности. Уровни срочности задаются в 1С и отображаются в выпадающем списке с настроенными иконками и цветами.

- Выберите один или несколько уровней срочности
- Кнопка **×** сбрасывает отбор
- Внизу меню — **Сбросить настройки**, чтобы вернуть иконки и цвета всех уровней к значениям по умолчанию

### Настройка иконок срочности

У каждого уровня срочности в выпадающем списке есть кнопка **⚙** (настройка). Нажмите на неё, чтобы открыть окно с выбором иконки и цвета для данного уровня.

Доступные иконки: двойная стрелка вверх, стрелка вверх, равно, стрелка вниз, флаг, молния, восклицательный знак.

![HEX и RGB](/img/UrgencyHexRgb.png)

После выбора нажмите **Сохранить** для применения. Закрыть окно без сохранения можно кнопкой **×**, кликом вне окна или клавишей **Escape**.

## Фильтр по типу карточки

![Фильтр по типу](/img/FilterType.png)

Фильтр **Тип** позволяет отобрать карточки по виду: **Задача** или **Ошибка**. За раз выбирается **один** тип; повторный клик по выбранному пункту снимает отбор.

Если на доске только задачи или только ошибки, фильтр **скрывается** автоматически.

![Скрытый фильтр типа](/img/HiddenFilters.png)

## Группировка

![Группировка](/img/Group.png)

Доска поддерживает группировку задач по двум критериям:

- **По исполнителю** — задачи группируются по исполнителям с отображением фотографии и имени

![Группировка по исполнителям](/img/ГруппировкаПоИсполнителю.png)

- **По проектам** — задачи группируются по проектам с отображением цвета проекта

![Группировка по проектам](/img/ГруппировкаПоПроектам.png)

Каждая группа содержит счётчик задач и может быть **свёрнута/развёрнута** нажатием на заголовок группы.

При группировке по проектам перетаскивание задач между группами запрещено. При группировке по исполнителям перетаскивание между группами меняет исполнителя задачи.

Для снятия группировки выберите **Без группировки**.

## Поиск по задачам

![Поиск](/img/Search.png)

Поле **Поиск по задачам...** фильтрует карточки по введённому тексту в реальном времени. Поиск выполняется по текстовому описанию задачи без учёта регистра. Для очистки поиска нажмите кнопку **×**.

## Переключатель темы

![Тёмная тема](/img/DarkTheme.png)

Кнопка переключения темы расположена в правой части панели управления. Поддерживаются **светлая** и **тёмная** темы. Выбранная тема сохраняется между сессиями.

## Добавление задачи или ошибки

Если в **[настройках](./userSettings.md#статусы)** доски для статуса установлена галочка **Кнопка добавить задачу**, в заголовке колонки этого статуса появляется кнопка **+**.

![Кнопка добавить](/img/Add.png)

По нажатию на **+** открывается форма создания в 1С. **Статус колонки подставляется автоматически** — не нужно выбирать его в форме.

### Когда тип выбирается сразу

Окно «Что создать?» **не появляется**, если:

- на доске только **задачи** — сразу открывается форма задачи;
- на доске только **ошибки** — сразу форма ошибки;
- в фильтре **Тип** выбран один вид (Задача или Ошибка) — создаётся элемент выбранного типа.

### Окно «Что создать?»

Если на доске одновременно отображаются **и задачи, и ошибки**, а фильтр типа не задан, появляется окно с вопросом **Что создать?** и кнопками **Задача** и **Ошибка**.

![Выбор типа при создании](/img/CreateKindModal.png)

Закрыть окно без создания можно кнопкой **×**, кликом вне окна или клавишей **Escape**.

После выбора типа открывается соответствующая форма 1С

После записи новый элемент появляется на доске автоматически. Если этого не произошло, нажмите **Обновить** на панели управления.

## Drag & Drop

Перетаскивание карточек между колонками изменяет статус задачи или ошибки. При перетаскивании целевая колонка подсвечивается.

В режиме группировки Drag & Drop также работает:

- **Группировка по исполнителям**: можно перетаскивать карточки между группами — при этом меняется исполнитель задачи
- **Группировка по проектам**: перетаскивание между группами разных проектов запрещено — колонка подсвечивается **красным**

## Автолимит карточек

![Ещё N](/img/More.png)

Если в колонке слишком много карточек и они не помещаются по высоте экрана, лишние карточки автоматически скрываются. Внизу колонки появляется ссылка **Ещё N**, по нажатию на которую показываются все карточки.

## Обновление отображения задач на доске

![Обновление](/img/Update.png)

Доска **обновляется автоматически** при изменении задач и ошибок в 1С (запись, смена статуса и т.д.). Кнопка **Обновить** принудительно перестраивает HTML доски, если отображение по каким-то причинам не совпало с данными.
