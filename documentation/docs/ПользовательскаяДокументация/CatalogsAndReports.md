# Справочники и отчёты

Раздел **OneKanban** в интерфейсе 1С содержит не только канбан-доску, но и справочники для ведения данных и отчёт для анализа времени в статусах.

![Команды подсистемы](/img/Commands.png)

Настройка колонок доски, проектов и сопоставления статусов описана в **[Настройках доски](./userSettings.md)**. Работа с фильтрами и карточками на доске — в **[Работе с доской](./OneKanban.md)**.

## Справочники

### Задачи

Основной объект учёта работ. Задачи попадают на доску, если их статус и проект входят в настройки доски.

Для встроенных объектов OneKanban в конфигурациях на БСП действует иерархия «проект → эпик → подзадачи» — см. **[Схему работы](./diagramBSP.md)**.

Открыть задачу можно из списка **Задачи** в подсистеме или **кликом по номеру на карточке** доски.

![Открытие задачи по ссылке](/img/CardLinkOpen.png)

### Ошибки

Справочник ошибок используется вместе с **[сопоставлением статусов](./userSettings.md#сопоставление-статусов)**: ошибки отображаются в тех же колонках, что и задачи, с соответствующим статусом. На доске ошибки отличаются подписью «Ошибка №…» и красной полосой слева на карточке.

### Проекты и статусы

- **Проекты** — группировка задач; цвет проекта задаётся в настройках доски
- **Статусы задач** и **Статусы ошибок** — колонки доски и значения для сопоставления

## Отчёт «История задач»

Отчёт **История задач** (`канбан_ИсторияЗадач`) показывает, сколько времени задачи и ошибки провели в каждом статусе. Данные накапливаются при записи элементов (регистр истории статусов).

Отчёт доступен в подсистеме **OneKanban** — команда **История задач** (раздел «История задач» в вариантах отчётов БСП).

![Отчёт — детальный вариант](/img/ReportHistory.png)

### Варианты отчёта

| Вариант | Содержание |
|---------|------------|
| **Детально** | Поэтапная история смены статусов по каждой задаче |
| **Сводка по задаче** | Суммарное время в каждом статусе в разрезе задачи |
| **Сводка по статусу** | Суммарное время, проведённое задачами в каждом статусе |
| **Сводка по исполнителю** | Время по исполнителям и статусам |
| **Сводка по проекту** | Время по проектам и статусам |

:::info

Состав полей и отборов зависит от конфигурации-носителя. В нестандартных внедрениях текст запроса отчёта может переопределяться в модуле `канбан_КанбанДоскаПереопределяемый`.

:::
