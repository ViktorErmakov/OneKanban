Как собрать проект:

1. В VSC открыть папку OneKanban-html
2. Запустить команду **npm i** в терминале для установки всех зависимостей
3. Настроить **webpack.config.js** при необходимости
4. Запустить команду **npm run build** в терминале для сборки единого файла HTML.

5. Замена макетов для юнит тестов:
- запускаем на отладку сам тест, поставив точку остановки в "ТекстHTMLКанбанДоски"
- заменяем в полученном тексте страницы "/Справочник.канбан_Задачи" на "/%1"
- сохраняем в макет текст страницы в одну строчку, без форматирования, так получает 1С страницу
